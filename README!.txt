Welcome to the C Strings Tutorial!

Prerequesites:
In order to follow this tutorial, you to have the following
preprocessor defined in the Visual Stuido Solution: _CRT_SECURE_NO_WARNINGS
In the Visual Studio Solution provided, this is already done.

This tutorial shows how to make an array of "char" (characters), to arrange
the characters into a word, phrase, or sentence. This is called a "string" of text,
because all the characters are in a line, like a string, to make words.
